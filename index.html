SportLFC — Data Systems & Database App

Public-safe package

A SQL and data-systems project for a youth sports league. A normalized relational schema — 14 core tables plus a PlayerFees view — backs role-aware Player, Coach, and Admin workflows, with seed data and analytics queries across registration, rosters, equipment, coaches, and fees. Built on PHP/MySQL with mysqli prepared statements and validation for rosters, equipment, and coach assignments, plus reporting queries for popular sports, roster utilization, coach workload, fee summaries, and equipment revenue. Synthetic demo data — not a production app.
